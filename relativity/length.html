幾何起源

桿在 S′ 中靜止，兩端點 x′ = 0、x′ = L₀ 對應兩條 平行於 ct′ 的世界線。

長度的定義是「兩端點 同時 量到的位置差」。S′ 用 ct′ = const 切片得固有長 L₀；S 用 ct = const 切片，因切片角度不同，量到的兩線距離較短。 L = L₀ √(1 − β²) = L₀ / γ
